University Student Details System

A full-stack web application developed using JSP and Servlets to manage university student details. The system allows admin login, student registration, searching by name or degree, and dynamic display of student records.

Features:
  • Admin authentication and session management
  • Student data entry and update
  • Search functionality by student name and degree
  • Dynamic and responsive data tables
  • Photo upload and display for students
Tech Stack: JSP, Servlets, MySQL, HTML, CSS, JavaScript
