Train Booking System

A web-based train ticket booking system allowing users to search trains, book tickets, and manage reservations. The system features user-friendly interfaces for passengers and admin controls for managing bookings.

Features:
  • Search and select trains by route and date
  • Ticket booking and confirmation
  • User registration and login
  • Admin dashboard for booking management
  • Responsive design for desktop and mobile
Tech Stack: JSP, Servlets, MySQL, HTML, CSS, JavaScript
