Scoob Run - Scooby-Doo Themed UI/UX Design

A fun and colorful Scooby-Doo themed mobile UI/UX design created for a fictional app called "Scoob Run". This project focused on engaging visuals, character-based navigation, and playful animations to enhance the user experience. Designed with beginners and casual users in mind.

Features:
  • Character-themed UI elements
  • Mobile-first layout and components
  • Custom icons, buttons, and interactions
  • Animated transitions and screens
  • Kid-friendly UX focus
Tools Used: Figma, Canva, Adobe XD (for mockups & design assets)
