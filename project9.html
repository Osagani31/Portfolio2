Library Management System

A web-based Library Management System that allows users to browse books, reserve, and manage returns. Admins can update book availability and monitor reservation status. Built with JSP, Servlets, and MySQL.

Features:
  • Book browsing and search
  • Book reservation and return management
  • Admin controls for status updates
  • User authentication and profile management
  • Responsive design
Tech Stack: JSP, Servlets, MySQL, HTML, CSS, JavaScript
